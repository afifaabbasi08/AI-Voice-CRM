# AI Voice CRM

An AI-powered CRM platform for field sales teams that converts WhatsApp voice notes into structured business records.

## Features Planned

- Voice Note Upload
- Speech-to-Text (Whisper)
- AI Data Extraction
- CRM Database
- Store Tracking
- Follow-Up Management
- Analytics Dashboard
- WhatsApp Integration

## Tech Stack

- Django
- PostgreSQL
- Python
- Faster-Whisper
- Chart.js
- GitHub
